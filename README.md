# 📝 Smart To-Do Web App

A modern and responsive To-Do Web Application developed using HTML, CSS, and JavaScript.

This application allows users to manage their daily tasks efficiently by adding, editing, deleting, and marking tasks as completed. Tasks are stored using Local Storage so data remains available even after refreshing the browser.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Set Date & Time for tasks
- Task statistics dashboard
- Responsive design
- Local Storage support
- Modern and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API


### 3. Run the Application

---

## 📋 How to Use

1. Enter a task in the input field.
2. Select a date and time.
3. Click the **Add** button.
4. Use the:
   - ✅ Complete button to mark a task as done
   - ✏️ Edit button to modify a task
   - 🗑️ Delete button to remove a task
5. View task statistics at the top of the page.

---

## 💾 Local Storage

The application uses the browser's Local Storage feature to save tasks permanently.

Tasks remain available even after:

- Refreshing the page
- Closing the browser
- Reopening the application
